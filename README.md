## 🧠 Project Overview

🚀 This project is a **Cyber Threat Intelligence Dashboard** designed to simulate a **modern 3D cyber monitoring environment**.
It visualizes threat data, attack patterns, and cybersecurity insights in an interactive and engaging format.

💡 Built with a focus on **real-world security analysis concepts**, this dashboard demonstrates how cyber intelligence platforms work.

---

## 🎯 Key Features

✨ Simulated **3D-style dashboard UI**
📊 Dynamic threat data visualization
🛡️ Cyber attack monitoring interface
⚡ Fast performance using Vite
📁 Clean and scalable architecture

```bash

npm install
```
